Списки элементов

9 сентября 3027-го года

Одна из самых базовых возможностей - отрисовывание списков. В Jetpack Compose нет специального функционала для этого, используется функция Column { ... }, рисующая свои элементы сверху вниз. Благодаря тому что во всём Compose интерфейс описывается обычным кодом на Kotlin, можно использовать циклы, if-ы и прочие фичи обычных языков программирования.

Column {
    for (i in 1..5)
        Text("List Item $i")
}

Красивый текст

9 сентября 3027-го года

Возможность красиво выделять и форматировать текст - одна из самых базовых и самых нужных в любом UI-фреймворке. HTML позволяет удобно стилизовывать отдельные фрагменты текста, не меняя всего стиля целиком, в Jetpack Compose же это API немного неудобное и проще менять стиль всего текста сразу

Форматирование текста

Инструменты стилизации текста
Отрендеренный HTML тег Его аналог в Compose
Выделенный жёлтым маркером текст 
Text(
    "Выделенный жёлтым маркером текст",
    style = TextStyle(background = Color.Yellow)
)
Жирный текст 
Text("Жирный текст", fontWeight = FontWeight.Bold)
Текст в италике 
Text("Текст в италике", fontStyle = FontStyle.Italic
Перечёркнутый текст 
Text(
    "Перечёркнутый текст",
    style = TextStyle(textDecoration = TextDecoration.LineThrough)
)
Подчеркнутый текст 
Text(
    "Подчёркнутый текст",
    style = TextStyle(textDecoration = TextDecoration.Underline)
)

Прямых аналогов тегов <del> и <ins> в Compose нет, но их можно реализовать самому
Типо удалённый текст 
Text(
    "- Типо удалённый текст",
    style = TextStyle(textDecoration = TextDecoration.LineThrough, color = Color.Red)
)
Типо добавленный текст 
Text(
    "+ Типо добавленный текст",
    style = TextStyle(textDecoration = TextDecoration.Underline, color = Color.Green)
)
ТекстЯ снизуЯ сверху 
Text(
    AnnotatedString(
        "ТекстЯ снизуЯ сверху",
        spanStyles = listOf(
                    Range(SpanStyle(baselineShift = BaselineShift.Subscript), 5, 12),
                    Range(SpanStyle(baselineShift = BaselineShift.Superscript), 12, 20),
        )
    )
)
Ссылка на сообщение в чате курса Фронтенда. 
Text(
    "Ссылка на сообщение в чате курса Фронтенда.",
    style = TextStyle(color = Color.Blue, textDecoration = TextDecoration.Underline)
)
